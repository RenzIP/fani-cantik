# **1. PENDAHULUAN** 

## **1.1 Tujuan** 

Dokumen ini disusun sebagai panduan penggunaan (user manual) bagi pengguna Aplikasi Inventaris Nasi Bakar, khususnya pada bidang atau modul Pesanan. Tujuan dari dokumen ini adalah untuk membantu pengguna, terutama kasir, memahami langkah-langkah dalam melakukan input pesanan pelanggan, memantau status pesanan, hingga melihat riwayat transaksi, sehingga proses pencatatan pesanan nasi bakar dapat berjalan lebih cepat, rapi, dan minim kesalahan. 

## **1.2 Gambaran Umum Produk** 

Aplikasi Inventaris Nasi Bakar adalah aplikasi berbasis web yang digunakan untuk mengelola operasional penjualan nasi bakar, mulai dari pencatatan menu, input pesanan pelanggan, pemantauan transaksi harian, hingga laporan pendapatan. Aplikasi ini dirancang agar mudah digunakan oleh kasir di lapangan dengan tampilan yang sederhana dan ringkas. 

Secara umum, aplikasi ini memiliki beberapa modul utama, yaitu: 

- Dashboard — menampilkan ringkasan operasional harian dan bulanan. 

- Kasir & Pemesanan — modul utama untuk melakukan input pesanan pelanggan. 

- Menu Nasi Bakar — menampilkan daftar menu, harga, dan deskripsi produk. 

- Riwayat Transaksi — menampilkan daftar transaksi yang telah dan sedang diproses. 

- Laporan — menampilkan ringkasan laporan transaksi dan pendapatan. 

Panduan ini secara khusus berfokus pada modul Pesanan, yaitu proses input pesanan pelanggan melalui halaman Kasir & Pemesanan. 

Aplikasi ini hanya memiliki satu peran pengguna, yaitu Kasir. Pengguna dengan peran Kasir memiliki akses penuh untuk masuk ke sistem, menginput pesanan pelanggan, memantau riwayat transaksi, serta melihat laporan penjualan sebagaimana dijelaskan pada bagian selanjutnya. 

# **2. SPESIFIKASI** 

## **2.1 Kebutuhan Sistem** 

- Perangkat: komputer, laptop, tablet, atau smartphone. 

- Koneksi internet yang stabil. 

- Web browser terkini (contoh: Google Chrome, Microsoft Edge, atau Mozilla Firefox). 

- Akun pengguna dengan peran Kasir yang sudah terdaftar pada aplikasi. 

## **2.2 Fitur Utama Modul Pesanan** 

Pada modul Kasir & Pemesanan, kasir dapat melakukan hal-hal berikut: 

- Melihat ringkasan pesanan hari ini, pendapatan hari ini, pesanan baru, dan total menu aktif. 

- Menginput data pelanggan (nama pelanggan). 

- Memilih metode pembayaran (contoh: Tunai). 

- Menentukan status pembayaran (contoh: Belum Bayar / Sudah Bayar).
